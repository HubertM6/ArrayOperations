Zadanie Lista Wartości
=================

W pliku `liczby.txt` podane jest n (1 <= n <= 100000) liczb z zakresu zmiennej typu int. Napisz program, dający odpowiedzi do poniższych zadań. Uzyskane odpowiedzi wypisz w konsoli lub zapisz w pliku `wyniki.txt`.

### Zadanie 1
Podaj maksymalną wartość.

### Zadanie 2
Podaj minimalną wartość.

### Zadanie 3
Podaj sumę wartości.

### Zadanie 4
Podaj średnią wartość. Uwaga: podaj wynik z **dokładnie** 4 miejscami po przecinku, uwzględniając ewentualne zera na końcu.

### Zadanie 5
Podaj medianę wartości. Uwaga: podaj wynik z **dokładnie** 4 miejscami po przecinku, uwzględniając ewentualne zera na końcu.
